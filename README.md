# Homebrew Tap

Custom Homebrew formulas by [@montanaflynn](https://github.com/montanaflynn).

## Install

```sh
brew tap montanaflynn/tap
```

## Formulas

| Formula | Description | Install |
|---------|-------------|---------|
| [cctop](https://github.com/montanaflynn/cctop) | TUI for monitoring Claude Code sessions | `brew install montanaflynn/tap/cctop` |
| [fdu](https://github.com/montanaflynn/fdu) | Fast disk usage analyzer with a streaming TUI | `brew install montanaflynn/tap/fdu` |
